# ESP32, Arduino + SOG128128A_T112 128 x 128 Transparent OLED (1.12")
Transparent graphic OLED module (SH1107 controller).

The module is sold by [Surenoo](https://www.surenoo.com/products/detail/23109435) and is also commonly listed on AliExpress.

## Hardware overview
- Display: 1.12" transparent graphic OLED, 128x128 pixels
- Controller: SH1107 (or equivalent)
- Interface: SPI
- Logic supply: 3.3V
- Panel supply: Vpp = 15V
- Reference current: Iref = 16.625uA, Rref = 750K
- ESP32 running an Arduino sketch

## ESP32 wiring (SPI)
- SCK: GPIO 18
- MOSI: GPIO 23
- CS: GPIO 16
- D/C (A0): GPIO 17
- RST: GPIO 21

## Firmware sketch
`SOG128128A_Firmware.ino`: video receiver for UART streaming
